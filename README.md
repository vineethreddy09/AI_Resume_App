# 📄 AI Resume Analyzer
This app uses AI to help users analyze their resumes and extract key information.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://airesumeapp.streamlit.app/)

## 🚀 How to use
1. Click the **Streamlit** button above to open the app.
2. Upload your Resume in **PDF** format.
3. The AI will automatically extract your contact info, skills, and experience!

## 🛠️ Built With
* [Streamlit](https://streamlit.io/) - The web framework
* [Pdfminer.six](https://github.com/pdfminer/pdfminer.six) - PDF parsing tool
* Python
